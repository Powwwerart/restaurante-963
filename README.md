# Restaurante QR Menu

Sistema minimalista de menú QR bilingüe (ES/EN) con precios claros y sin imágenes.

## Requisitos
- Node.js opcional solo si quieres servir archivos localmente con un servidor estático.

## Instalación
No requiere dependencias.

## Ejecución
Sirve el proyecto con cualquier servidor estático. Ejemplo con `npx`:

```bash
npx serve .
```

Luego visita:

- `/r/restaurante-963`

## Variables de entorno
No se requieren.
